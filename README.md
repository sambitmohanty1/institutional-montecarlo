# Institutional Monte Carlo + AI Stock Analyst

Unified Streamlit app for institutional-grade investment research.

## Features
- **Single Stock Mode:** Technicals, fundamentals, and AI-style valuation hints.
- **Portfolio Mode:** Monte Carlo simulations on multiple tickers.
- **Outputs:** CAGR, confidence percentiles, interactive charts.

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Cloud
- Push to your GitHub repo.
- Go to https://share.streamlit.io
- Select `app.py` as the entry point.
